Система управления ингредиентми

    Проект состоит из четырёх классов:
        Ingredient описывает отдельные ингредиенты с измерениями и операции над этими ингредииентами;
        Recipe собирает и обрабатывает рецепты - списки, состоящие из ингредиентов;
        DietaryRecipe - его дочерний класс, дополнительно хранит тип диеты, к которой относится тот или иной рецепт;
        ShoppingList хранит список кортежей - пар из ингредиентов и рецептов, к которым они принадлежат, и обрабатывает     соответсвующие операции;
    И тестов к этим классам.


Для установки необходимо клонировать репозиторий и установить зависимости:
    git clone https://github.com/ar1zonaranger/HW2
    cd HW2
    pip install -r requirements.txt
Тесты запускаются командой 
        python -m pytest test_recipes.py -q
